# Seminar Hub 2026

**Seminar Hub 2026** adalah aplikasi Android modern yang dirancang untuk mengelola dan mendaftar seminar teknologi. Aplikasi ini menonjolkan desain antarmuka (UI) yang futuristik dengan tema gelap (*Dark Mode*), efek *Glassmorphism*, dan aksen merah elektrik yang elegan.

## 📺 Video Preview

Lihat demo aplikasi di sini: [Video Preview Seminar Hub 2026](https://drive.google.com/file/d/1eKzgYsS80QL0X461vudgwhvOHReQMwOe/view?usp=drive_link)

## 🚀 Fitur Utama

- **Modern UI/UX**: Menggunakan desain *Sleek Rounded Cards*, gradasi halus, dan tipografi kontras tinggi.
- **Glassmorphism Design**: Implementasi efek kaca transparan pada header dan kartu konten untuk estetika premium.
- **Seminar Dashboard**: Daftar seminar interaktif dengan informasi topik, pembicara, dan jadwal tahun 2026.
- **Form Pendaftaran Valid**: Sistem pendaftaran dengan validasi *real-time* (Nama, Email, Nomor HP) untuk memastikan akurasi data.
- **Responsive Layout**: Optimal untuk berbagai ukuran layar perangkat Android.

## 🛠️ Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: XML dengan [Material Components](https://material.io/develop/android)
- **Architecture**: ViewBinding & Material Design 3 (Glassmorphism style)
- **Navigation**: Jetpack Navigation Component
- **Support**: Android API 24+

## 🎨 Palet Warna

- **Deep Charcoal**: Latar belakang utama yang memberikan kesan profesional.
- **Electric Red**: Digunakan sebagai aksen *Call-to-Action* (CTA) dan elemen penting.
- **Soft Grey & Glass White**: Memberikan tekstur dan kedalaman pada elemen UI.

## 📸 Tampilan Aplikasi

- **Login Screen**: Akses masuk dengan desain kartu transparan.
- **Dashboard**: Panel utama menampilkan seminar unggulan (Modern Android, UI/UX, AI Integration).
- **Registration Form**: Formulir pendaftaran lengkap dengan pilihan seminar dinamis.
- **Result Screen**: Ringkasan data pendaftaran setelah berhasil disubmit.

## ⚙️ Cara Instalasi (Menggunakan ZIP)

Jika Anda mendapatkan proyek ini dalam format file ZIP:

1. **Ekstrak File**: Klik kanan pada file `FormLogRegiss.zip` dan pilih **Extract Here** atau **Extract to Folder**.
2. **Buka di Android Studio**:
   - Buka Android Studio.
   - Pilih **File > Open**.
   - Arahkan ke folder hasil ekstrak tadi dan klik **OK**.
3. **Tunggu Sync**: Biarkan proses **Gradle Sync** selesai (pastikan koneksi internet aktif untuk mendownload library yang diperlukan).
4. **Jalankan Aplikasi**: Klik tombol **Run** (segitiga hijau) atau tekan `Shift + F10`.
